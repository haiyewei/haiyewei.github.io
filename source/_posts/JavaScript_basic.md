## <center>JS 基础

### 函数
结构: function 函数名(传递参数) {函数内部语句}
```js
function name() {
    语句1;
    语句2;
}
```
### 变量常量
结构: var\let\const 变量名;
```js
var name;
let age;
const height = 165; #必须初始化
```

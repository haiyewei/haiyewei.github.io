## <center>Markdown 基本语法

### [标题]
```markdown
#一级标题
##二级标题
###三级标题
......最多六级
```

### [引用]
```markdown
>"引用内容"
```

### [有序列表]
```markdown
1. 内容
2. 内容
3. 内容
```
### [无序列表]
```markdown
- 内容
- 内容
- 内容
* 内容
* 内容
```

### [任务列表]
```markdown
- [x] 内容 ("x"标记已完成项目)
- [ ] 内容
- [ ] 内容
- [ ] 内容
```

### [代码块]
```markdown
-------------------------------------------
```c

printf("这是一段代码");

```
-------------------------------------------
```

### [数学公式]
```markdown
$$
"公式内容"
$$
```

### [表格]
```markdown
|姓名|年龄|成绩|    (表头)
|:---|---:|:---:|   (对齐方式)
|张三|18|122|
```

### [脚注]
```markdown
文本内容[^脚注标号] 

[^脚注标号]:脚注内容
```

### [横线插入]
```
---[输出一条横线]
```

### [超链接]
```markdown
[描述文本](链接 "标题")
[百度](baidu.com "搜索引擎")
```

### [引用链接]
```markdown
[描述文本][链接标号]

[链接标号]:链接 "标题"
```

### [设置标题链接]
```markdown
[描述文本](#参考的标题名)
```

### [URL生成]
```markdown
URL:
输入标准链接地址会自动生成超链接
```

### [导入图片]
```markdown
![描述文本](图片链接)
```

### [文本行内格式]
```markdown
*斜体*
**加粗**
`行内代码`
<u>下划线</u>
:输入表情名称可输出标准表情符号:
~下标~
^上标^
==高亮文字==
```
